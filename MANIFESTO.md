# The GateCrash Manifesto

## We Crash Gates. We Don't Build New Ones.

## The Problem

Big Tech has become the gatekeeper of everything digital:

- **Google Forms** bans AI agents, even from paid accounts
- **TypeForm** charges $50/month for basic features
- **SurveyMonkey** locks your data behind paywalls
- **Jotform** requires their branding on free plans
- **Mailchimp** demands compliance with their arbitrary rules
- **AWS** controls your infrastructure and pricing

The pattern is clear: **Build a tool ‚Üí Get users ‚Üí Lock them in ‚Üí Extract rent**

They say it's about "quality" or "security" or "terms of service."

**We say it's gatekeeping.**

---

## The GateCrash Philosophy

### 1. BYOK (Bring Your Own Keys)

You own the infrastructure. Not us. Not anyone else.

- Your SMTP server for emails
- Your storage for data
- Your deployment for hosting
- Your API keys for integrations

**If we don't control it, we can't gatekeep it.**

### 2. Toolmaker, Not Service Provider

GateCrash is NOT a SaaS platform. We don't want your:
- Credit card
- User data
- Monthly subscription
- Dependency

We build **tools**, not **services**. Tools that YOU control.

### 3. Open Source Forever

Every line of code is open. Every feature is inspectable. Every decision is transparent.

You can:
- Fork it
- Modify it
- Sell it
- Host it yourself
- Never pay us a cent

**The code is yours. Forever.**

### 4. No Lock-In

Export your data anytime. Switch tools anytime. Host anywhere. Use any SMTP server.

We use **standard formats**:
- JSON for configs
- HTML for forms
- CSV/JSON for responses
- SMTP for email

No proprietary formats. No vendor lock-in.

## Why Now?

The internet was supposed to be **decentralized**.

Instead, we got:
- 3 companies controlling 90% of cloud infrastructure
- 2 companies controlling mobile operating systems
- 1 company controlling search
- Walled gardens everywhere
- APIs that cost $10,000/month
- "Enterprise plans" for basic features

**This is not freedom. This is feudalism.**

And now they're gatekeeping **AI agents** too. Google bans agent usage. OpenAI restricts automation. Anthropic monitors "unusual activity."

**They're terrified of us automating away their control.**
